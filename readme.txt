Woocommerce Marketplace Product Mass Upload allows you to uplaod products in mass number using csv.

== Description ==

This plugin will allows admin and seller to upload products in mass number using csv. Admin and seller both can upload all type of products woocommerce provided like simple, external, etc. Even admin can bulk upload into Seller account using the CSV file.

== Installation ==

1. Install the Marketplace Product Mass Upload
2. Activate the plugin
3. Go to the Mass Upload Page
